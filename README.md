# Portfólio | Kendra Riquena 💻

Bem-vindo ao repositório do meu portfólio pessoal! Este projeto foi desenvolvido para apresentar minhas habilidades, experiências e projetos como Desenvolvedora Web Full Stack.

O site é totalmente responsivo, moderno e focado na experiência do usuário.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias e ferramentas:

*   **HTML5**: Estrutura semântica da página.
*   **CSS3**: Estilização personalizada, uso de variáveis CSS (`:root`) e Flexbox.
*   **Bootstrap 5**: Framework utilizado para o sistema de grid, navbar responsiva e componentes de interface.
*   **Font Awesome**: Ícones para redes sociais e ilustrações visuais.
*   **Google Fonts**: Fonte "Catamaran" para a tipografia do site.

## 🎨 Design e Estilo

O layout segue uma paleta de cores moderna e vibrante:
*   **Cor Primária**: `#FF2E63` (Rosa)
*   **Cor Secundária**: `#252A34` (Cinza Escuro)
*   **Fundo**: Variações de cinza claro e branco para contraste.

## 📂 Estrutura das Seções

1.  **Home**: Apresentação inicial com chamada para ação (CTA).
2.  **Sobre Mim**: Resumo profissional e soft skills (Código Limpo, Criatividade, Trabalho em Equipe).
3.  **Habilidades**: Listagem técnica dividida em Frontend, Backend e Ferramentas.
4.  **Projetos**: Vitrine de trabalhos recentes (Ex: Resimetal, Zaions, Global Supply Automation).
5.  **Contato**: Links diretos para E-mail, WhatsApp e LinkedIn.

## 🔧 Como executar o projeto

1.  Clone este repositório:
    ```bash
    git clone https://github.com/Kendra-Riquena/NOME-DO-REPOSITORIO.git
    ```

3.  Abra o arquivo `index.html` no seu navegador de preferência.
